# Introduction to Programming in R: A Workshop Series 💻

Welcome! This repository was created as part of the 3-part "Introduction to Programming in R" workshop series, part of the **Advancing Reproducibility with Computational Skills (ARCS)** initiative at the Yale Medical Library.

This series is designed to guide you from the fundamentals of R programming to more advanced topics, including data manipulation with the Tidyverse and version control with Git/GitHub for collaborative and reproducible research.

------------------------------------------------------------------------

## Part 1: Fundamentals and Project Management

### Summary

If you are new to coding or have never used R before, this workshop is for you! In this introduction to R fundamentals, you will complete hands-on coding exercises, follow along with more complex R scripts, and connect these skills to reproducible research practices.

### In this workshop, you will learn to:

-   Navigate the RStudio interface and execute basic R commands
-   Apply fundamental programming concepts (objects, functions, basic operations)
-   Create vectors, data frames, and matrices
-   Find a dataset in a public repository and read dataset files into RStudio
-   Implement file organization and naming conventions for reproducible research
-   Understand the advantages of R for research workflow automation

### Prerequisites:

-   A computer where you have the necessary permissions to install software
-   [R](https://cran.r-project.org/) and [RStudio](https://posit.co/download/rstudio-desktop/) installation

------------------------------------------------------------------------

## Part 2: Data Structures and Reports

### Summary

If you want to start learning how to use R to analyze data and be able to report your findings, this workshop is for you! Using the same dataset from Part 1, you will learn how R stores data, how to make choices as you set up your data for analysis, and how to create a reproducible record of your work.

### In this workshop, you will learn to:

-   Manipulate vectors, lists, matrices, and data frames
-   Identify the advantages and limitations of different data structures in R
-   Implement literate programming principles in data analysis
-   Generate dynamic reports using Quarto for reproducible documentation

### Prerequisites:

-   Completion of Part 1: Fundamentals and Project Management
-   [Quarto](https://quarto.org/docs/get-started/) installation

------------------------------------------------------------------------

## Part 3: Tidyverse Packages and Git

### Summary

If you want to start leveraging the R packages ecosystem while keeping track of how you do it, this workshop is for you! Using the same dataset from Parts 1 & 2, you will learn how to transform and "tidy" your data using the Tidyverse, and extend your code management skills by learning to collaborate effectively using Git for version control.

### In this workshop, you will learn to:

-   Understand how to subset data effectively
-   Execute complex data transformations using `dplyr` (`filter`, `select`, `mutate`, `summarize`, `group_by`) and other useful packages
-   Initialize Git repositories and commit project changes
-   Understand version control and collaboration methods using GitHub

### Prerequisites:

-   Completion of Part 2: Data Structures and Reports
-   [GitHub Account](https://github.com/signup)
-   [GitHub Desktop](https://desktop.github.com/)
-   Windows users only: [Git Bash](https://git-scm.com/downloads/)
-   **Note:** Please accept all defaults for installer prompts.
